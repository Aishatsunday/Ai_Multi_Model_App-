# AI Multi-Model App

A **Streamlit web application** that combines **CIFAR-100 image classification**, **credit card fraud detection**, and **customer segmentation** into a single, easy-to-use interface.

---

## Features

1. **CIFAR-100 Image Classifier**
   - Upload an image and classify it into one of 100 categories.
   - Uses a fine-tuned **ResNet-50** model.

2. **Credit Card Fraud Detection**
   - Predict whether a transaction is **fraudulent or legitimate**.
   - Input transaction details and get instant results.

3. **Customer Segmentation**
   - Group customers based on behavioral features.
   - Helps with marketing and business decisions.
